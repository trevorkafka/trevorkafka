Logarithms

In this section, $b$ is used to represent any positive number other than $1$, often referred to as "the base."

Definition

Logarithms ($\log$) form a family of functions of various bases defined so that the following two forms are equivalent and convey the same information.

Exponential Form

$$a = b^c$$ $$\text{“$a$ equals $b$ to the $c$"}$$

Logarithmic Form

$$c = \log_b a$$ $$\text{“$c$ equals $\log$ base $b$ of $a$"}$$
note In $\log_b a$, despite its appearance, $a$ is not an exponent on $b$. The variable $b$ is what is called the base of the function $\log_b$ and $a$ is the input of that function $\log_b$.

Logarithms as Inverses

$$ \cancel b^{\cancel{\log_b} x} = x \text{ for $x > 0$}$$ $$\bcancel \log_b {\bcancel b}^x = x \text{ for all $x$}$$ In other words, if $f(x) = b^x$, then the inverse function is $f^{-1}(x) = \log_b x$.
proofs
proof of first statement Pick a positive value of $x$ and let $c$ be the unique solution to the equation $x = b^c$ (the requirement that $x>0$ is a necessary condition for this equation to have a unique solution). Converting to logarithmic form gives $$c = \log_b x.$$ Substituting this value of $c$ then gives $$x = b^{\log_b x}.$$
proof of second statement Pick any value of $x$ and let $c = b^x$. Converting to logarithmic form gives $$x = \log_b c.$$ Substituting in $b^x$ for $c$ gives $$x = \log_b b^x.$$
proof of inverses Let $f(x) = b^x$ and $g(x) = \log_b x$. Since $$f(g(x)) = b^{\log_b x} = x$$ and $$g(f(x)) = \log_b b^x = x,$$ $f$ and $g$ must be inverse functions, so $g = f^{-1}$.
tricky examples $$\log_b 1 = 0$$ $$\log_b b = 1$$

Special Notation

$$\log x \text{ means } \log_{10} x$$ $$\ln x \text{ means } \log_e x$$ $$e = 2.718281828\ldots$$
note $\ln$ is rerpesents what is called the "natural logarithm." $e$ is known as "Euler's number" or "the base of the natural logarithm," which is an irrational number. Both $\ln$ and $e$ come up frequently in calculus in order to model particular situations with simple constraints on rates of change.

Domain and Range

$y=\log_b x$ has a domain of $x > 0$ and range of all $y$ values.
proof It is known that $f(x) = b^x$ has a domain of all $x$ values and a range of $y>0$. Since $f(x)$ is one-to-one, the inverse function $f^{-1}(x) = \log_b x$ thus has the "swapped" version of this domain and range.
note Another way of interpreting the domain is to say$$\log_b \left( \begin{matrix} \text{zero or any} \\ \text{neg. number} \end{matrix} \right)$$ is always undefined.

Multiplication Property

$$\log_b xy = \log_b x + \log_b y$$ $$\text{ for $x,y>0$}$$
proof Pick any positive values of $x,y$ (this is a requirement for $\log_b x$ and $\log_b y$ to be defined). Note that by the exponent property $z^{m+n} = z^m \cdot z^n$, we have $$b^{\log_b x + \log_b y} = \bcancel b^{\bcancel{\log_b} x} \cdot \bcancel b^{\bcancel{\log_b} y},$$so $$b^{\log_b x + \log_b y} = xy,$$which when converted to logarithmic form gives $$\log_b xy = \log_b x + \log_b y.$$
warning In general, the rule does not work in the following backwards sense: $$\color{red} \log_b \left( x + y \right) \neq (\log_b x )( \log_b y)$$ In fact, $\log_b \left( x + y \right)$ is generally not simplifiable.

Division Property

$$\log_b \left( \frac{x}{y} \right) = \log_b x - \log_b y$$ $$\text{ for $x,y>0$}$$
proof Pick any positive values of $x,y$ (this is a requirement for $\log_b x$ and $\log_b y$ to be defined). Note that by the exponent property $z^{m-n} = \frac{z^m}{z^n}$, we have $$b^{\log_b x - \log_b y} = \frac{\bcancel b^{\bcancel{\log_b} x} }{ \bcancel b^{\bcancel{\log_b} y}},$$so $$b^{\log_b x - \log_b y} = \frac{x}{y},$$which when converted to logarithmic form gives $$\log_b \left( \frac{x}{y} \right) = \log_b x - \log_b y.$$
warning In general, the rule does not work in the following backwards sense: $$\color{red} \log_b \left( x - y \right) \neq \frac{\log_b x}{\log_b y}$$ In fact, $\log_b \left( x - y \right)$ is generally not simplifiable. However, $\frac{\log_b x}{\log_b y}$ simplifies via the change of base rule: $$\frac{\log_b x}{\log_b y} = \log_y x$$

Exponent Property

$$\log_b (x^y) = y \log_b x$$ $$\text{ for $x>0$}$$
proof Pick values of $x,y$ such that $x>0$. Let $c = \log_b(x^y)$. Converting to exponential form gives $$b^c = x^y$$and raising both sides to the power of $\frac{1}{y}$ gives $$b^{\frac{c}{y}} = x.$$ Now, converting back to logarithmic form gives $$\frac{c}{y} = \log_b x,$$so $$\log_b(x^y) = c = y \log_b x.$$
tricky examples $$\log_b \frac{1}{x} = - \log_b x$$ $$\log_b \sqrt{x} = \frac{1}{2} \log_b x$$ $$\log_b \frac{1}{\sqrt x} = - \frac{1}{2} \log_b x$$ $$\log_b \sqrt[m]{x^n} = \frac{n}{m} \log_b x$$ $$\log_b \frac{1}{\sqrt[m]{x^n}} = -\frac{n}{m} \log_b x$$

Change of Base Rule

$$\log_{b_1} x = \frac{\log_{b_2} x }{\log_{b_2} b_1}$$ $$\text{for $x > 0$}$$ This identity demonstrates a conversion from a base $b_1$ to a new base $b_2$.

proof Pick any positive value $x$ and let $c = \log_{b_1} x$ ($x$ being positive is a requirement of $\log_{b_1} x$ being defined). Converting to exponential form gives $$b_1^c = x$$and taking $\log_{b_2}$ of both sides gives $$\log_{b_2} \left( b_1^c \right) = \log_{b_2} x,$$ $$c \cdot \log_{b_2} b_1 = \log_{b_2} x,$$ $$c = \frac{\log_{b_2} x}{\log_{b_2} b_1},$$so $$\log_{b_1} x = \frac{\log_{b_2} x }{\log_{b_2} b_1}$$.
note It also works for $\log$ and $\ln$ since these are just the special cases where $b=10$ and $b=e\approx2.718$, respectively: $$\log_{b_1} x = \frac{\log x}{\log b_1} = \frac{\ln x}{\ln b_1}$$ $$\text{for $x>0$}$$
