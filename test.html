Logarithms

In this section, $b$ is used to represent any positive number other than $1$, often referred to as "the base."

Definition

Logarithms ($\log$) form a family of functions of various bases defined so that the following two forms are equivalent and convey the same information.

Exponential Form

$$a = b^c$$ $$\text{“$a$ equals $b$ to the $c$"}$$

Logarithmic Form

$$c = \log_b a$$ $$\text{“$c$ equals $\log$ base $b$ of $a$"}$$
note In $\log_b a$, despite its appearance, $a$ is not an exponent on $b$. The variable $b$ is what is called the base of the function $\log_b$ and $a$ is the input of that function $\log_b$.

Logarithms as Inverses

$$ \cancel b^{\cancel{\log_b} x} = x \text{ for $x > 0$}$$ $$\bcancel \log_b {\bcancel b}^x = x \text{ for all $x$}$$ In other words, if $f(x) = b^x$, then the inverse function is $f^{-1}(x) = \log_b x$.
examples $$\log_b 1 = 0$$ $$\log_b b = 1$$

Special Notation

$$\log x \text{ means } \log_{10} x$$ $$\ln x \text{ means } \log_e x$$ $$e \approx 2.718281828$$
note $e$ is known as "Euler's number" or "the base of the natural logarithm," which is an irrational number.

Domain and Range

$y=\log_b x$ has a domain of $x > 0$ and range of all $y$ values.
note Another way of interpreting the domain is to say$$\log_b \left( \begin{matrix} \text{zero or any} \\ \text{neg. number} \end{matrix} \right)$$ is always undefined.
Graph of $y = \log_b x$
$$\log_b xy = \log_b x + \log_b y$$ $$\text{ for $x,y>0$}$$
$$\log_b \left( \frac{x}{y} \right) = \log_b x - \log_b y$$ $$\text{ for $x,y>0$}$$
$$\log_b (x^y) = y \log_b x$$ $$\text{ for $x>0$}$$
examples $$\log_b \frac{1}{x} = - \log_b x$$ $$\log_b \sqrt{x} = \frac{1}{2} \log_b x$$ $$\log_b \frac{1}{\sqrt x} = - \frac{1}{2} \log_b x$$ $$\log_b \sqrt[m]{x^n} = \frac{n}{m} \log_b x$$

Change of Base Rule

$$\log_{b_1} x = \frac{\log_{b_2} x }{\log_{b_2} b_1}$$ $$\text{for $x > 0$}$$ This identity demonstrates a conversion from a base $b_1$ to a new base $b_2$.

note It also works for $\log$ and $\ln$ since these are just the special cases where $b=10$ and $b=e\approx2.718$, respectively: $$\log_{b_1} x = \frac{\log x}{\log b_1} = \frac{\ln x}{\ln b_1}$$ $$\text{for $x>0$}$$
