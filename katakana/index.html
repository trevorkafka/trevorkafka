Katakana Reader

Welcome to Katakana Reader, an internet tool that will allow you to practice your katakana reading abilities. Simply indicate which katakana you would like to practice and you will be shown a randomized list of words that use those specific katakana. Add new katakana once you feel ready.

Select Katakana

Which katakana do you want to practice? Here, toggle each katakana that you would like to practice by tagging it as "allow," "emphasize," or "require." Select the type of tag below and then click the kana you would like to apply the tag to. Click the arrows to apply the tag to entire rows at once.
